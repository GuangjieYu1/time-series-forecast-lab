# Frontend Fixtures

The backend fixture generator creates the canonical sample files under `backend/tests/fixtures`.
Frontend acceptance testing can use those files directly:

- `daily_air_passengers.csv`
- `monthly_air_passengers.xlsx`
- `raw_flight_detail_multi_sheet.xlsx`
- `invalid_date.csv`
- `duplicate_dates.xlsx`
- `missing_values.xlsx`
- `short_series.csv`
- `legacy_daily_air_passengers.xls`
